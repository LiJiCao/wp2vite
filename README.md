# wp2vite

一款转化工具，可以一键让webpack项目支持vite

tips：目前暂时只支持react

# 安装
```
npm install -g wp2vite
```
# 使用
```
cd yourwork/your_project
wp2vite init

npm install

npm run dev
or
npm run vite-dev
```
